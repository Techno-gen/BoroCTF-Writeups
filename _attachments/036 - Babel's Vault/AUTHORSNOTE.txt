This library is one that is held very dear to me. When I was younger, I was obsessed with ciphers like Ceaser Cipher, Vigenere, and XOR. Today however, those are far from secure, with more modern encryption like RSA taking the stand. Faced with this, the older ciphers have become used more as an aesthetic than for actual purpose. The neat part is however, that you can still use these, and if you do it well enough, people will not notice. Still, the methods I previously mentioned are too common and recognizable to actually make a purpose of, so inventing new methods is more desirable. Creation of an actual cipher like this is nearly impossible, yet I have set myself towards achieving this goal. I became aware of the Library of Babel, and before trying to make a cipher with it, I had decided to learn how it works by creating this library. What you see now is the finished product, and I hope you can have fun trying it out. Thanks
